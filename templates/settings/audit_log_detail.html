Field glossary (for humans & AI)

  • event_code — Stable identifier; filter the list or grep. http.* = middleware capture of a mutating request; other prefixes (submissions., settings., email.) are explicit app events.
  • outcome — For middleware rows, derived from HTTP status. Explicit logs usually Success.
  • human_summary — Plain-English one-liner; use in incident descriptions.
  • correlation_id — UUID for one browser request; multiple rows may share it if several log calls occur in the same request.
  • details — JSON: path, method, status, query params, optional post_keys_summary (values truncated; passwords/tokens redacted).
